## Installation

- **Requirements**:
  - **Node.js**: >= 16.0

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Raffian-moin/React-hook-form-lecture.git

2. **Navigate to the project directory:**
    ```bash
    cd React-hook-form-lecture

4. **Install dependencies:**
    ```bash
    npm install

5. **Run project:**
    ```bash
    npm run dev
